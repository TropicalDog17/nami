# Data Models

Domain entities and data structures for transaction tracking and financial management.

## Components

- **`transaction.go`** - Transaction entity with validation and derived field calculations
- **`transaction_type.go`** - Transaction type definitions and categories
- **`investment.go`** - Investment entities and portfolio management structures
- **`fx_rate.go`** - Foreign exchange rate data structures
- **`asset_price.go`** - Asset price and market data models
- **`master_data.go`** - Master configuration data entities
- **`reporting.go`** - Reporting data structures and analytics models
- **`actions.go`** - Workflow triggers and action entities
- **`ai_pending_action.go`** - AI processing queue entity

Defines domain models, validation rules, and database schema mappings for all financial data.
