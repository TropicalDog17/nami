# Data Access Layer

Repository pattern implementation for database operations and query optimization.

## Components

- **`interfaces.go`** - Repository interface definitions and contracts
- **`transaction_repository.go`** - Transaction CRUD operations and queries
- **`investment_repository.go`** - Investment data access and portfolio queries
- **`reporting_repository.go`** - Optimized reporting queries and analytics
- **`ai_pending_action_repository.go`** - AI processing queue management

Provides database query execution, persistence, retrieval, and transaction management with optimized query patterns.
