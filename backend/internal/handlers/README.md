# HTTP Handlers

REST API endpoint implementations for all transaction tracking operations.

## Components

- **`transaction.go`** - Transaction CRUD operations and validation
- **`admin.go`** - Administrative operations and system management
- **`reporting.go`** - Financial reporting and analytics endpoints
- **`fx.go`** - Foreign exchange rate management
- **`investment.go`** - Investment operations and management
- **`vault.go`** - Vault management operations
- **`price.go`** - Asset price and market data endpoints
- **`pending_actions.go`** - AI processing queue management
- **`action.go`** - Workflow triggers and actions

Handles HTTP request parsing, validation, response formatting, and error status codes for all API endpoints.
