# Backend API Server

TypeScript REST API server with PostgreSQL database for transaction management.

## Structure

- **`src/core/`** - Core business logic and domain models
- **`src/handlers/`** - HTTP request handlers
- **`src/services/`** - Business service layer
- **`src/repositories/`** - Database access layer
- **`src/database/`** - Database connection and schema
- **`src/monitoring/`** - Metrics and health checks
- **`src/scripts/`** - Utility scripts
- **`src/utils/`** - Helper utilities
- **`src/index.ts`** - Application entry point
- **`src/types.ts`** - TypeScript type definitions
- **`src/openapi.ts`** - OpenAPI/Swagger specification
- **`tests/`** - Test suite
