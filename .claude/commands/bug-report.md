Here is my description of the issue:
[Insert your bug description here]

Using my description, generate a bug report that includes:

Bug Title

Summary

Severity & Priority (propose both)

Steps to Reproduce

Expected Result

Actual Result

Screenshots/Logs Section (leave placeholders if not provided)
