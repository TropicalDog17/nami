# Frontend Integration Tests

Integration tests for end-to-end communication with the backend.

## Tests

- Vault CRUD operations
- Vault share management
- Vault transaction processing
- Summary and reporting endpoints
- Error handling and validation
- Data consistency and immutability
- Decimal precision

## Running Tests

```bash
cd frontend
npm run test:integration
```

Prerequisites: Backend running on http://localhost:8080 with migrations applied.
