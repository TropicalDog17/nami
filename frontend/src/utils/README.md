# Utility Functions

Shared utility functions and helper modules for common operations and data manipulation.

## Components

- **Data transformation** - Functions for converting and formatting financial data
- **Validation helpers** - Input validation and data sanitization utilities
- **Formatting utilities** - Currency, date, and number formatting functions
- **Calculation functions** - Financial calculations and mathematical utilities
- **Date/time utilities** - Date manipulation and timezone handling
- **Configuration constants** - Shared constants and configuration values

Provides reusable helper functions for data processing, validation, formatting, and common calculations throughout the application.
