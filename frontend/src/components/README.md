# React Components

Reusable UI components organized by function.

## Structure

- **`forms/`** - Form components for transaction and data entry
- **`modals/`** - Modal dialog components and overlays
- **`reports/`** - Financial reporting and data visualization
- **`ui/`** - Basic UI utility components and design system

## Key Components

- **`Layout.tsx`** - Main application layout and navigation
- **`VaultManager.tsx`** - Vault management interface
- **`AdminAssetsTab.tsx`** - Asset management interface
- **`CreditCardSummary.tsx`** - Credit card transaction display
- **`FloatingAddButton.tsx`** - Floating action button

Provides modular, reusable components with consistent design patterns.
