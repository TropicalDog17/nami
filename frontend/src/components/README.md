# React Components

Reusable UI components organized by function for building the transaction management interface.

## Structure

- **`forms/`** - Form components for transaction and data entry
- **`modals/`** - Modal dialog components and overlays
- **`reports/`** - Financial reporting and data visualization components
- **`ui/`** - Basic UI utility components and design system

## Key Components

- **`Layout.tsx`** - Main application layout wrapper and navigation
- **`AdminAssetsTab.tsx`** - Admin interface for asset management
- **`VaultManager.tsx`** - Vault management and operations interface
- **`CreditCardSummary.tsx`** - Credit card transaction summary display
- **`FloatingAddButton.tsx`** - Floating action button for quick actions

Provides modular, reusable components with consistent design patterns and user interaction handling.
