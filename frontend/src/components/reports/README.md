# Reporting Components

Data visualization and financial reporting components for analytics and insights.

## Components

- **`Charts.tsx`** - Generic chart components and visualization utilities
- **`AssetAllocationChart.tsx`** - Asset allocation portfolio visualization
- **`CashFlowChart.tsx`** - Cash flow trend analysis and charts
- **`PnLDisplay.tsx`** - Profit and loss summary and display

Provides interactive data visualization, financial reporting interfaces, and chart configuration for transaction analytics.
