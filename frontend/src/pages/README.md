# Page Components

Route-level React components implementing the main application interfaces and user workflows.

## Components

- **`TransactionPage.tsx`** - Transaction management and entry interface
- **`AdminPage.tsx`** - Administrative interface for system configuration
- **`ReportsPage.tsx`** - Financial reporting and analytics dashboard
- **`VaultsPage.tsx`** - Vault overview and management interface
- **`VaultDetailPage.tsx`** - Detailed vault view and operations
- **`CreditDashboardPage.tsx`** - Credit card spending dashboard

Implements page-specific business logic, data presentation, user interactions, and navigation integration for all major application routes.
