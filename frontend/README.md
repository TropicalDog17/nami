# Frontend

React web application for transaction management and financial reporting.

## Structure

- **`src/components/`** - React components organized by feature (ui, forms, modals, reports)
- **`src/pages/`** - Application pages and routes
- **`src/hooks/`** - Custom React hooks
- **`src/services/`** - API client and data services
- **`src/context/`** - React context providers
- **`src/utils/`** - Helper utilities
- **`src/styles/`** - Global styles and Tailwind configuration
- **`tests/`** - End-to-end and component tests
