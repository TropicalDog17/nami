# Frontend

React web application for transaction management and financial reporting with modern UI.

## Structure

- **`src/`** - React application source code
- **`tests/`** - End-to-end and component testing suite
- **`public/`** - Static assets and entry HTML

## Key Features

Responsive design with Tailwind CSS, transaction management interface, comprehensive reporting dashboards, admin configuration panels, and real-time data synchronization with the backend API.
